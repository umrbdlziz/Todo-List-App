
# Todo-List-App
A simple todo list app using React Native



## Features

- Add Task
- Delete Task


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Create new app

```bash
  npx todolist --template
```

Go to the project directory

```bash
  cd todolist
```

Start the server

```bash
  npx expo start
```
