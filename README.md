
# Todo-List-App
A simple todo list app using React Native



## Features

- Add Task
- Delete Task


## Run Locally

Clone the project

```bash
  git clone https://github.com/umrbdlziz/Todo-List-App.git
```

Create new app

```bash
  npx create-expo-app todolist
```

Go to the project directory

```bash
  cd todolist
```

Start the server

```bash
  npx expo start
```
